# Tic-Tac-Toe

Backend implementation for Tic-Tac-Toe game.

## Features to Implement
- Game board management
- Player management
- Move validation
- Win condition checking
- Game state management

## Package Structure
- `models/` - Domain models (Board, Player, Move, etc.)
- `service/` - Game logic and business rules
- `strategy/` - Different game strategies (winning algorithms)
- `exception/` - Custom exceptions
