# Splitwise

Backend implementation for expense sharing application.

## Features to Implement
- User management
- Group management
- Expense management
- Split strategies (Equal, Exact, Percentage)
- Balance calculation
- Simplified debt settlement

## Package Structure
- `models/` - Domain models (User, Group, Expense, Split, etc.)
- `service/` - Business logic
- `strategy/` - Split strategies
- `algorithm/` - Balance simplification algorithms
- `exception/` - Custom exceptions
