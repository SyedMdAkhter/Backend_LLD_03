# BookMyShow

Backend implementation for movie ticket booking system.

## Features to Implement
- Theater and screen management
- Movie management
- Show management
- Seat booking and reservation
- Payment processing
- User management

## Package Structure
- `models/` - Domain models (Theater, Screen, Movie, Show, Seat, Booking, etc.)
- `service/` - Business logic
- `repository/` - Data access layer
- `exception/` - Custom exceptions
