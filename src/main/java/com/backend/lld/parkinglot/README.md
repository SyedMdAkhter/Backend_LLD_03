# Parking Lot

Backend implementation for Parking Lot management system.

## Features to Implement
- Multiple parking floors
- Different vehicle types (Car, Bike, Truck)
- Different parking spot types
- Parking spot assignment strategies
- Ticket generation and payment
- Entry/Exit management

## Package Structure
- `models/` - Domain models (ParkingLot, Floor, Spot, Vehicle, Ticket, etc.)
- `service/` - Business logic
- `strategy/` - Parking strategies (nearest first, etc.)
- `exception/` - Custom exceptions
